# Sistema de Gestiòn de Activos y Prestamos de Equipos AIT-Control- SIISCA


📦 IT Asset Management System (Backend)

Este es un sistema de gestión de activos tecnológicos diseñado para optimizar el control de hardware y software en entornos corporativos o industriales. El proyecto se enfoca en la trazabilidad de equipos, asignación de usuarios y generación de reportes automatizados.
🚀 Características Principales

    Gestión de Inventario (CRUD): Registro completo de dispositivos (Laptops, Monitores, Periféricos) vinculados a una base de datos MySQL.

    Control de Asignaciones: Seguimiento en tiempo real de qué usuario posee cada activo.

    Generación de Reportes: Exportación automática de datos a formatos legibles como Excel/CSV utilizando Python.

    Contenedores con Docker: Despliegue simplificado para asegurar que el entorno sea consistente en cualquier servidor.

🛠️ Tecnologías Utilizadas

    Lenguaje: Python.

    Base de Datos: MySQL / PostgreSQL.

    Contenedores: Docker.

    Control de Versiones: Git y GitHub.

    Análisis de Datos: Librerías para manipulación de archivos y optimización de código.

  📋 Requisitos e Instalación

    Clonar el repositorio: git clone https://github.com/tu-usuario/it-asset-management.git

    Construir la imagen de Docker: docker-compose up --build.

    Acceder a la API/Interfaz para gestionar los activos.
